## 💡 Por que o Ebyte Manager?

O Manager oficial da Evolution API não permite que o usuário final visualize suas próprias instâncias, tokens e informações importantes. O **Ebyte Manager** resolve esse problema com uma interface simples e completa.

### ✨ Funcionalidades Exclusivas

- 👁️ **Visualização Completa** - Veja TODAS as informações da sua instância
- 🔑 **Token Visível** - Copie facilmente o token para suas automações
- 📊 **Estatísticas Reais** - Quantidade de conversas, contatos e mensagens
- 💬 **Envio Direto** - Teste mensagens sem precisar de outra ferramenta
- 🎨 **Interface Moderna** - Bootstrap 5 com tema claro/escuro
- 📱 **100% Responsivo** - Funciona perfeitamente no celular
