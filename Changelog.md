# Changelog - Ebyte Manager API

## [1.0.0] - 2024-01-XX
### Funcionalidades
- Conexão via QR Code
- Gerenciamento de instância única
- Envio de mensagens (texto e mídia)
- Suporte a múltiplos destinatários
- Intervalo aleatório entre envios
- Tema claro/escuro
- Logs em tempo real
- Sessão persistente (24h)

### Segurança
- Token CSRF
- Validação de inputs
- Headers de segurança
